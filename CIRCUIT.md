## Connections

- `SDA` and `SCL` between Arduino and Raspberry Pi with pull-ups; common `GND`.

## Diagram (ASCII)

```
 Arduino SDA <----> RPi SDA (pull-up)
 Arduino SCL <----> RPi SCL (pull-up)
 GND --------------- Common ground
```
